# UI Specification

이 문서는 웹 UI 요구사항 초안입니다.
현재 기본 방향은 Streamlit 기반이지만, 구현 기술은 상황에 따라 변경할 수 있습니다.
다만 아래 기능 요구사항과 데이터 흐름은 유지합니다.

## 1. UI 범위
- 대상 온실: `gh1`, `gh2`
- 주요 탭
  - 실시간 모니터링
  - 액추에이터 제어
  - 과거 추세

## 2. 공통 정책
- 화면 어디서든 현재 날짜/시간 표시
- 각 페이지(탭)별로 "마지막 업데이트 시간" 1개 표시
- 각 센서값마다 개별 업데이트 시간을 표시하지 않음
- 온실은 2개지만 각각의 온실에 라즈베리파이와 아두이노, 디스플레이가 설치되어 있어 독립적으로 동작함

## 3. 데이터 흐름
### 3.1 권장 구조
- 센서/상태 원천 데이터는 MQTT(JSON)로 수집됨
- logger 서비스가 MQTT를 받아 SQLite에 저장
- UI는 SQLite의 최신값/과거값을 읽어 표시
- UI의 제어 입력은 MQTT command로 publish
- KMA API 데이터도 logger를 통해 DB에 저장된 값을 표시하는 구조를 권장

### 3.2 DB 초안 메모
- 최신값 조회용 뷰 이름은 `docs/db-schema.md` 초안을 참고합니다.
- 현재 초안에는 아래 latest 뷰가 정의되어 있습니다.
  - `latest_sensor_snapshot`
  - `latest_weather`
  - `latest_actuator_state`
  - `latest_heartbeat`
  - `latest_fan_rpm`
- 다만 위 이름은 DB 스키마 최종 확정 전까지 변경될 수 있습니다.

### 3.3 구현 메모
- 초기 MVP에서 UI와 저장 로직이 함께 있을 수는 있지만, 구조는 나중에 logger 분리가 가능하도록 설계하는 것을 권장

## 4. 탭별 요구사항

### 4.1 실시간 모니터링 탭
- 표시 대상
  - 센서 12채널
  - 외기 정보(KMA)
- 표시 방식 예시
  - 카드형, 표형, 섹션형 중 선택 가능
- 페이지 마지막 업데이트 시간 1개 표시
- 센서 항목
  - 온도 2개
  - 습도 2개
  - CO2 1개
  - PAR 1개
  - 토양수분 6개
- 외기 항목
  - 외기온도
  - 외기습도
  - 필요 시 지역명

### 4.2 액추에이터 제어 탭
- 목적
  - 사용자가 수동으로 액추에이터를 제어
  - 선택된 온실의 command topic으로 MQTT 발행

- 페이지 마지막 업데이트 시간 1개 표시

- 제어 항목
1. 레벨 제어(숫자 입력 + 슬라이더)
   - 환기팬 1
   - 히터 2
   - 순환팬 2
   - 펌프 1

2. ON/OFF 제어
   - 솔레노이드 밸브 7
     - 화분 6
     - 포깅 전단 1
   - 초음파 미스트 1

3. 창문 제어
   - 창문 2개
   - 명령: open / close / stop
   - 개도율은 시간 기반으로 계산
   - 초기 실행 시 10초 close 후 기준을 잡는 동작 포함

4. LED 제어
   - addressable LED
   - RGB + 밝기 슬라이더

5. Arduino 재부팅
   - Raspberry Pi GPIO를 사용해 3.3V 릴레이 NC 접점을 제어
   - 클릭 방식 제어 가능
   - press-and-hold는 필수 아님
   - 현재는 GPIO 릴레이 개별 테스트 스크립트가 존재하고, 운영용 백엔드 흐름은 별도 확정 필요

- 추가 표시
  - 각 팬 제어 항목 옆 또는 근처에 현재 RPM 표시
  - RPM은 Arduino가 publish한 값을 사용

### 4.3 과거 추세 탭
- 기간 선택 기능 필요
- 그래프 그룹
  - 온도 그래프: 내부 온도 2개 + 외기온도
  - 습도 그래프: 내부 습도 2개 + 외기습도
  - 토양수분 그래프: 6개 채널
- 페이지 마지막 업데이트 시간 1개 표시

## 5. 상태/에러 표시(권장)
- Arduino online/offline 표시 권장
- 마지막 actuator state 수신 여부 표시 권장
- KMA 데이터 미수신 시 경고 메시지 권장
