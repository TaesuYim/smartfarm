# Pin Map

이 문서는 현재 설계 기준 핀 맵을 정리합니다.
센서 입력은 ADS1115 주소/채널 기준으로, 액추에이터 출력은 Arduino UNO R4 WiFi 보드별 핀 기준으로 기록합니다.

## 1. ADS1115 입력 맵

| ADS1115 주소 | 채널 | 연결 대상 |
| --- | --- | --- |
| `0x4b` | `A0` | 온도 하부 |
| `0x4b` | `A1` | 습도 하부 |
| `0x4b` | `A2` | 온도 상부 |
| `0x4b` | `A3` | 습도 상부 |
| `0x49` | `A0` | CO2 |
| `0x49` | `A1` | 조도 |
| `0x49` | `A2` | 토양수분1 |
| `0x49` | `A3` | 토양수분2 |
| `0x48` | `A0` | 토양수분3 |
| `0x48` | `A1` | 토양수분4 |
| `0x48` | `A2` | 토양수분5 |
| `0x48` | `A3` | 토양수분6 |

## 4. 메모

- 표의 `PWM 사용`은 현재 설계에서 `~` 표기로 표시된 핀을 옮긴 것입니다.
- 이 문서는 핀 배치 기준 문서이며, 실제 전원선/드라이버/공통 GND를 포함한 배선도는 별도 문서로 정리합니다.
- MQTT 키 이름과 DB 필드 이름은 각각 `docs/naming-conventions.md`, `docs/db-schema.md`를 기준으로 맞춥니다.
